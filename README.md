# 📸 PixelScape

Welcome to the PixelScape Image Gallery project!

PixelScape is a modern and responsive image gallery developed using HTML, CSS, and JavaScript. It provides an immersive visual browsing experience through elegant layouts, interactive image viewing, smooth animations, and intuitive navigation controls.

The application allows users to explore a collection of images in a beautifully designed gallery interface. With features such as lightbox viewing, image navigation, keyboard controls, and responsive layouts, PixelScape ensures a seamless experience across desktops, tablets, and mobile devices.

This project demonstrates essential frontend development concepts including DOM manipulation, event handling, responsive web design, image handling, and interactive user interface development.
